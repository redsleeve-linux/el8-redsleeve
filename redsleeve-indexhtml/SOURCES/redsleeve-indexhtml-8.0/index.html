Welcome to RedSleeve

The Enterprise Operating System for ARMs

RedSleeve is an Enterprise-class Linux Distribution derived from sources freely provided to the public by Red Hat, Inc. for Red Hat Enterprise Linux. RedSleeve conforms fully with the upstream vendors redistribution policy and aims to be functionally compatible. (RedSleeve mainly changes packages to remove upstream vendor branding and artwork and fix ARM specific package bugs.)

RedSleeve is developed by a small team of core developers.  In turn the core developers are supported by an active user community including system administrators, network administrators, enterprise users, managers, core Linux contributors and Linux enthusiasts from around the world.

RedSleeve has mirrors, developers who are contactable and responsive, and multiple community support avenues including a git, Email Lists, and a Bugs Database.
